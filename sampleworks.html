NSFPatents

NSFPatents on Github

NSFPatents scrapes the USPTO patent grant XML repository for patents with mentions of National Science Foundation in the Government Interest Field. It the correlates the patent back to the NSF award via the award ID mentioned in the field. It uses Nokogiri for the xml and html parsing.

I began to develop a successor to the project, called NSFPatents2, which was unit tested via RSpec, but this project did not reach completion.
